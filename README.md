# 杭州电子科技大学研究生论文LaTeX模版

格式要求见《杭州电子科技大学研究生学位论文格式统一要求》（杭电研[2012]311号）

推荐的编译命令为`latexmk -xelatex -synctex=1 main.tex`。
`latexmk`命令的运行需要系统安装有Perl解释器。
可以使用命令`latexmk -C`来删除编译产生的文件，可以使用命令`latexmk -c`来删除编译产生的临时文件。
